rerun {metaSEM}R Documentation

Rerun models via mxRun()

Description

It reruns models via mxRun().

Usage

rerun(object, ...)

Arguments

object

An object of either class tssem1FEM, class tssem1REM, class wls, class meta or class reml.

...

Futher arguments to be passed to mxRun

Author(s)

Mike W.-L. Cheung <mikewlcheung@nus.edu.sg>
